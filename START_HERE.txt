WHAT THE TOOL DOES:
-------------------

Analyzes Ford FDRS diagnostic logs and provides:
  ✓ Root cause analysis for flash failures
  ✓ Software mismatch detection (which DIDs need updating)
  ✓ DTC analysis with Service Info links
  ✓ Critical event timeline with timestamps
  ✓ Built-in training (hex decoder, failure patterns)
  ✓ Smart deduplication (filters NRC-31 noise)

NEED HELP?
----------

1. Check README.md for comprehensive troubleshooting
2. Test with test4.txt to verify tool works

Common Issues:
  - "No module named tkinter" → Reinstall Python with "tcl/tk" checked
  - Analysis takes time → Normal for large logs (5-30 seconds)
  - Report shows "Unknown" → Ensure log is FDRS format, test with test4.txt
